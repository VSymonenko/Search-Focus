# Contribution

## Get started

- clone project
- install dependencies ```npm i```

## Environment

- python 2.7
- NodeJS 16.0.0

## Known issues

1. If you use more than one versions of python you need to configure path to current version

```bash
py --list-paths  # To see the installed Python versions
```

```bash
export NODE_GYP_FORCE_PYTHON="<path>"
```
